# Installation
- Download XenonLinux iso
- Flash iso into an usb
- Boot into usb
- Login
```
user: live
pass: live
```
- Or login with root account
```
user: su
pass: toor
```
- Run the installer
  - Open launcher by pressing `Mod + d`
  - Choose `Install system`
